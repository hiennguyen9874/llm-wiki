# Source - Fincept Terminal

## What this source is
A README / documentation source for `Fincept Terminal`, a native desktop financial intelligence platform with analytics, data connectors, broker integrations, AI agents, and workflow tooling.

## Why it matters
This source adds a durable platform reference to the trading cluster:
- a native C++20 / Qt6 desktop terminal
- embedded Python for analytics
- broad market and macro data connectivity
- real-time trading and broker routing claims
- QuantLib-style analytical modules
- a visual node editor for workflows
- optional alternative-data overlays
