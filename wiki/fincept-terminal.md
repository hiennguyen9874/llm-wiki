# Fincept Terminal

## Summary
Fincept Terminal is a native desktop financial intelligence platform: the README presents it as a C++20 / Qt6 app with embedded Python analytics, broad market and macro data connectors, broker integrations, AI agents, QuantLib-style modules, and a node editor for automated workflows.

The strongest durable takeaway is the platform shape. It is a Bloomberg-terminal-like desktop system that combines research, analytics, execution, and workflow automation in one native application rather than splitting them across multiple tools.

> [!note]
> The README is promotional and uses snapshot-style counts. Preserve the claims, but treat them as source-backed documentation rather than independently verified measurements.

## Key capabilities
### Architecture
- Pure native C++20 desktop application
- Qt6 for UI and rendering
- embedded Python for analytics and research logic

### Analytics and research
- CFA-level analytics framing
- DCF, portfolio, and risk-oriented feature set
- QuantLib suite of pricing / risk / volatility / fixed-income modules
- AI agents for analysis workflows

### Data and execution
- 100+ data connectors across market, macro, and government sources
- real-time crypto / equity / algo / paper trading workflows
- 16 broker integrations
- optional Adanos market sentiment overlays for equity research

### Workflow and visualization
- node editor for automation pipelines
- MCP/tool-style integration language in the marketing
- dashboard-style product imagery and workflow surfaces
