# Jesse

## Summary
Jesse is an advanced crypto trading framework for Python strategy authoring, backtesting, optimization, and live trading.

Its durable value is breadth: the README presents Jesse as an all-in-one trading workflow that combines strategy logic, technical indicators, validation, execution, monitoring, optimization, Monte Carlo testing, and ML-related tooling in one project.

> [!note]
> The source is promotional and product-focused. The safest wiki reading is that Jesse advertises these capabilities; code-level verification would be needed to upgrade every feature claim into a stronger fact.

## Key capabilities
### Strategy authoring
- concise Python strategy classes
- built-in technical-analysis helpers
- support for multiple timeframes and symbols
- simple syntax for order, entry, take-profit, and stop-loss logic

### Trading workflow
- backtesting
- live trading
- leveraged and short-selling support
- partial fills
- smart ordering across market, limit, and stop orders

### Risk and evaluation
- built-in risk management helpers
- metrics system
- debug mode
- Monte Carlo analysis for robustness checking

### Optimization and ML
- optimize mode for parameter search / tuning
- JesseGPT assistant claim
- built-in code editor
- machine-learning pipeline for feature gathering, training, and deployment
