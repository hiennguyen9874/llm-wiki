# Source - The Simple 4-Step Process to Build Your Own AI Trading Assistant With Claude

## What this source is
A Vietnamese written summary of a video about building a personal AI trading assistant with Claude Code. The emphasis is on a repeatable workflow: first plan the assistant, then build a skeleton, then personalize it to the trader’s real process, and finally wire it into daily routines.

## Why it matters
This source expands the vault’s Claude Code trading cluster beyond execution bots and TradingView tooling. The durable lesson is not a particular dashboard implementation; it is the build workflow:
- plan before coding
- ask structured questions instead of improvising too early
- build a local dashboard that reflects the trader’s own process
- capture tendencies, coach notes, and journaling data
- make the assistant part of a daily routine
