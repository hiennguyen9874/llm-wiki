# Raw Sources

This directory holds immutable source documents for the second brain.

## Subfolders
- `inbox/` — drop new sources here before ingestion
- `articles/` — saved articles and essays
- `books/` — book notes, excerpts, chapter files
- `papers/` — papers and technical reports
- `web-clips/` — markdown from web clippers
- `captures/` — quick captures, fleeting notes, copied snippets, and rough observations
- `assets/` — downloaded images and attachments
- `codes/` — source codes

## Rule
Do not modify the contents of source documents after they are placed here.
Interpretation, normalization, crystallization, and synthesis belong in `wiki/` and `outputs/`.
If a source contains sensitive information, sanitize it before it enters this repository.
