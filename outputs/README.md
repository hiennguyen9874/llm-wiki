# Outputs

Derived artifacts generated by the LLM live here.

## Subfolders
- `answers/` — durable answers to user questions
- `analyses/` — comparisons, memos, research notes
- `briefings/` — reusable summaries prepared for later reference or sharing
- `crystallizations/` — digests distilled from completed research, debugging, or exploratory sessions
- `reports/` — lint reports, reviews, audits

Promote especially valuable outputs back into `wiki/` as stable pages.
Crystallizations are the bridge between exploratory work and durable semantic memory.
