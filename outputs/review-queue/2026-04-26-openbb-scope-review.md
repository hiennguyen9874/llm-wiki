# OpenBB synthesis scope review

## Decision needed
Should [[openbb]] remain a standalone project page, or later fold into a broader `financial-data-infrastructure` synthesis if more similar sources arrive?

## Context
The OpenBB README is a substantial documentation snapshot for an open-source financial data platform and enterprise Workspace surface. It expands the wiki’s finance cluster upstream into data integration, analyst workspaces, MCP, and REST/API surfaces.

## Recommendation
Keep OpenBB as a standalone project page for now.

## Options
1. **Keep standalone** — best if this remains the only major OpenBB-like data platform source.
2. **Fold into synthesis later** — best if additional financial data infrastructure / workspace sources arrive and the cluster starts to fragment.
