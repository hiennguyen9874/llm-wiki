# Ingest Plans

Selective storage for stage-1 ingest plans when they have durable audit value: non-trivial integration decisions, contradictions, review items, or broad page-update plans.

Plans should include source identity, related pages, key entities/claims, proposed updates, contradictions/supersession, review items, and whether overview/index/log/visual artifacts need updates.
